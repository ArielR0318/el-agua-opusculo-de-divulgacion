# Página Web de Divulgación sobre el Agua

Proyecto desarrollado en colaboración con la **BUAP** (Benemérita Universidad Autónoma de Puebla).

## Descripción
Sitio web informativo y educativo sobre la importancia del agua, diseñado con un enfoque responsive y moderno.

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript (Vanilla)

## Funcionalidades implementadas
- Diseño completamente responsive
- Optimización de rendimiento
- Interactividad con JavaScript
- Mejoras de usabilidad y accesibilidad

## Mi contribución

- Implementación de efectos y funcionalidades interactivas
- Pruebas de funcionamiento y optimizaciones
- Presentación final del proyecto ante investigadores
